Advanced network intrusion detection system that monitors network traffic in real-time to identify malicious activities.

**Tech Stack**: Python, Scapy, Machine Learning

**Key Features**:

- Anomaly detection using pattern recognition

### PasswordChecker
[View Repository](https://github.com/VanilPatel/PasswordChecker)

Comprehensive password strength analyzer that evaluates security based on multiple factors and industry best practices.

**Tech Stack**: Python, Regex, Entropy Calculations

**Key Features**:
- Strength scoring based on multiple criteria
- Common password and breach database checking
- Complexity analysis (character variety, length, patterns)
- Real-time feedback and improvement suggestions
- Dictionary attack simulation

Advanced phishing detection and prevention tool designed to protect users from malicious websites and fraudulent domains.

**Tech Stack**: Python, Machine Learning, URL Analysis

**Key Features**:

- Content analysis and pattern matching
- SSL certificate validation

- Heuristic analysis for zero-day phishing attempts
- High accuracy detection rate for phishing domains
